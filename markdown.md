## Caracteres de escape
\*Asteriscos Literales\*
Markdown proporciona caracteres de escape para:
the following characters:
\ backslash
` backtick
* asterisk
_ underscore
{} curly braces
[] square brackets
() parentheses
# hash mark
+ plus sign
- minus sign (hyphen)
. dot
! exclamation mark
